# Sherry (Ke) Shi

Personal site, live at **[kkkkshi.github.io](https://kkkkshi.github.io/)**.

- [`index.html`](https://kkkkshi.github.io/) : homepage. Who I am, experience, projects, papers.
- [`footprints.html`](https://kkkkshi.github.io/footprints.html) : an interactive antique-chart world map tracing everywhere I've lived and traveled, 1999 to now.

Plain HTML/CSS/JS with no build step. The map is rendered with self-hosted [MapLibre GL JS](https://maplibre.org/), published via GitHub Pages.
