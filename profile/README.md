# profile/

Content + assets for the homepage (`/index.html`).

- **`data.js`** — all homepage text (name, about, experience, education, projects,
  skills, featured GitHub repos). Edit this to update the site; nothing else needs touching.
- **`avatar.jpg`** *(optional)* — a photo, if you want one in the hero later.
- **`Ke_Shi_Resume.pdf`** *(optional)* — drop a PDF here and set `links.resume`
  to `"profile/Ke_Shi_Resume.pdf"` in `data.js` to show a “Résumé” download button.
  (Left off for now — the docx includes a phone number, so decide before making it public.)

The map lives at `/footprints.html`; its data stays in `/data`, `/audio`, `/photos`.
