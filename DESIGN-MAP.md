# 足迹地图设计规范 — `footprints.html`

> `footprints.html` 是「**古海图**」主题作品，**主题独立**：羊皮纸 + 衬线字 + 罗盘 + 做旧边框。
> 它**不套**首页（[DESIGN.md](DESIGN.md)）的系统字体 / grouped-gray 灰阶——那是有意为之的氛围。
> 但把首页规范里的**结构规则**（材质 / 层级 / 按钮态 / 圆角 / 暗色禁纯黑 / 无障碍）叠在 **UI 浮层**上，像 Apple Maps 在地图上浮卡片。
> 自检：[`tests/design-map.js`](tests/design-map.js)。

## 0. 主题基调（有意为之，不动）

| token | 值 | 用途 |
| --- | --- | --- |
| `--paper` | `#ece3cf` | 羊皮纸底 |
| `--ink` / `--ink2` | `#3a2f23` / `#6b5a44` | 正文主/次 |
| `--gold` | `#b89344` | 描边 / 分隔 |
| `--marker` | `#9b2d20` | 印章红 / 强调 |
| 字体 `--fell` | `"IM Fell English", "Songti SC", serif` | 古海图衬线 |

罗盘、印章、做旧四边框、海图填色——主题元素，保持。

## 1. 套用的结构规则（来自 DESIGN.md）

### §材质 / 透明度
- 信息卡片 `#card` = **半透明羊皮纸 + `backdrop-filter: blur(20px)`**（Apple Maps 式磨砂浮层）。
- 年份带 `#years` 底层 = `backdrop-filter: blur(9px)` 径向遮罩浮层。

### §层级 Elevation
- z-index：地图 `1` < 做旧/光照 `2–3` < 浮层（年份/面包屑/按钮/印章）`6–7` < **信息卡片 `8`**。
- 卡片分层投影：内高光 + 羊皮纸描边环 + 柔和落影。

### §形状 Shape
- 卡片圆角 **10px**、`next` 按钮 **7px**、圆形控件 50%。**不直角**。

### §按钮 & 交互态
- 所有可点元素都有 `hover` + **`:active` 按压态**：卡片 `next` / 音乐 / 昼夜 / 年份按钮。

### §文字不透明度分级
- 年份带：未选 **55%** → hover **85%** → 选中 **100%**。
- 正文两级：`--ink`（主）/ `--ink2`（次）。

## 2. 昼夜模式（暗色规则）

- `#daynight` 切换 `body.night`。
- 夜晚底色 = 深 navy **`#0f1826`**（**非纯黑**，符合 DESIGN.md §3.1 禁纯黑），文字 / 标签转浅金，配合地图深色。

## 3. 无障碍

- `prefers-reduced-motion` 时关闭开场虹彩 / 过渡动画。
