## 4. 干什么（任务阶梯，按性价比自排）

服务北极星。每轮挑**一件**收益高、风险低、不需拍板的事。

- **A. 填充式 ArcGIS 城市图（重点）**：让程序化地图像真的 ArcGIS Pro 绘图——**街区一块块填色**（深浅分层）、主干道 / 次街道、海岸线 + 海湾、河流、轻噪点 / 暗角。关键是**「有填充、看得到城市肌理」**，不是纯线条。每座城**不同且可信**。
- **B. 风格变体供对比（重点）**：出**几种配色 / 密度方案**（暖橙 / 冷调 / 浅淡、街区密疏…），让用户一眼挑「哪种效果好」（切换或并排画廊）。
- **C. 干净展示（不是小红书 App）**：地图放进**画廊 / 网格平铺**；**别做 feed / 帖子 / 底部 tab / captions / 点赞**那套。城市名这种最小标识可留。
- **D. 移动端**：手机视口下地图**清晰、可滚、不崩、不溢出**（390 / 430）。
- **E. 健壮 / 性能**：0 console 报错、SVG 别拖垮滚动、`prefers-reduced-motion`、对比度。
- **F. 代码质量**：只在 `mock-rednote.html` 内去重 / 简化。不碰别的文件。

> 全做完、测试全绿、剩下都需用户拍板 → `LOOP-LOG-REDNOTE.md` 写总结（列待决项），**停，不再调度下一轮**。别反复刷「没事可做」。
